# Cities Game (Java Swing)

Навчальний Java-проєкт — графічна гра **«Міста»**, реалізована з використанням **Java Swing**, архітектури **MVC** та збірки через **Gradle**.

---

## 📌 Опис гри

Гравець і комп’ютер по черзі називають міста.

- Кожне наступне місто має починатися з **останньої значущої літери** попереднього
- Повторювати міста не можна
- Гра триває, поки:
  - **Гравець не здається** (вводить `здаюсь`) → перемагає комп’ютер
  - **Комп’ютер не може знайти місто** → перемагає гравець

---

## 🧠 Правила обробки слів

- Ігноруються наприкінці слова: `ь`, `й`, `ъ`, пробіли, дефіси, апостроф
- Назви міст нормалізуються (регістр, апостроф)
- База міст: **Україна + Європа**

---

## 🏗 Архітектура

Проєкт побудований за патерном **MVC**:

```
model       — бізнес-логіка гри
view        — Swing-вікна
controller  — звʼязок UI ↔ логіка
```

## 🚀 Запуск проєкту

### Вимоги
- Java **17+**
- Gradle (або Gradle Wrapper)

### Запуск
```bash
./gradlew run
```

Windows:
```bat
gradlew.bat run
```
